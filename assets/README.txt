PLACEHOLDER ASSETS
==================

This folder should contain the following image files:

1. hero-background.jpg - Hero section background (1920x1080)
2. video-placeholder.jpg - Video section background (1920x1080) 
3. avatar-1.jpg - Testimonial avatar 1 (200x200)
4. avatar-2.jpg - Testimonial avatar 2 (200x200)
5. avatar-3.jpg - Testimonial avatar 3 (200x200)
6. icon.png - App icon (512x512)
7. splash.png - Splash screen (1242x2436)
8. favicon.png - Web favicon (48x48)

To generate placeholder images:
1. Open placeholder-generator.html in a web browser
2. Right-click each canvas and save as the corresponding filename
3. Or use any image editor to create placeholder images

For now, the app will show warnings about missing images but will still run.
